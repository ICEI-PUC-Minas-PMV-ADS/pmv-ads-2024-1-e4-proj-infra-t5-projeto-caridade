# CHARITY FINDER

`Análise e desenvolvimento de sistemas`

`Projeto`

`Eixo 4`

Buscador de projetos de caridade que tem como objetivo auxiliar na divulgação e busca de informações relativas a instituições filantrópicas espalhadas globalmente.
